# **혼자 공부하는 머신러닝+딥러닝**  

## **개요**  
이 레포지토리는 **2023년 8월 5일**부터 **12월 20일**까지 **혼자 공부하는 머신러닝+딥러닝** 교재를 기반으로 머신러닝과 딥러닝의 기초 개념을 정리한 것입니다. 파이썬을 활용해 데이터를 불러오고 모델을 학습·평가하는 전반적인 과정을 공식 예제 코드를 중심으로 학습하였습니다.

## **주요 특징**  
- 이론 설명과 함께 공식 예제 코드 실습 중심
- 복잡한 수식보다는 모델 학습 과정 이해에 집중
- 데이터 분석 이후 단계로서의 AI 입문 학습 기록


## **기술 스택**  
- **Python**
- **scikit-learn**
- **TensorFlow**
- **Keras**
